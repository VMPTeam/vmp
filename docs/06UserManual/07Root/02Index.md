# 2  首页 #
平台管理员登录系统后，进入系统首页，系统首页是对服务器的监控，分为服务器状态监控和系统监控。

## 2.1  服务器状态监控
服务器状体监控包括：在线用户数，应用服务器的网络延迟，数据库的网络延迟、活动的连接数和最大连接数，Redis服务器、Kafka对了服务器的情况统计，具体统计情况如下图所示：
